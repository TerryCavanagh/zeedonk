package terrylib.bitmapFont

class BitmapTextField

extends Sprite

Class for rendering text with provided bitmap font and some additional options.

Constructor

new (?font:BitmapFont, ?text:String, ?pixelSnapping:PixelSnapping, ?smoothing:Bool)

Constructs a new text field component.

Parameters:

font

optional parameter for component's font prop

text

optional parameter for component's text

Fields

alignment:BitmapTextAlign

Specifies how the text field should align text.

autoSize:Bool

Whether this text field have fixed width or not. Default value if true.

autoUpperCase:Bool

Whether to convert text to upper case or not.

background:Bool

Specifies whether the text should have background

backgroundColor:UInt

Specifies the color of background in 0xAARRGGBB format.

borderColor:UInt

The color of the border in 0xAARRGGBB format

borderQuality:Float

How many iterations do use when drawing the border. 0: only 1 iteration, 1: one iteration for every pixel in borderSize A value of 1 will have the best quality for large border sizes, but might reduce performance when changing text. NOTE: If the borderSize is 1, borderQuality of 0 or 1 will have the exact same effect (and performance).

borderSize:Float

The size of the border, in pixels.

borderStyle:TextBorderStyle

Use a border style

font:BitmapFont

Font for text rendering.

letterSpacing:Int

The distance to add between letters.

read only lineHeight:Float

Height of the single line of text (without lineSpacing).

lineSpacing:Int

The distance to add between lines.

multiLine:Bool

Specifies whether the text field will break into multiple lines or not on overflow.

read only numLines:Int

Reflects how many lines have this text field.

numSpacesInTab:Int

Number of space characters in one tab.

padding:Int

Number of pixels between text and text field border

read only shadowOffset:Point

Offset that is applied to the shadow border style, if active. x and y are multiplied by borderSize. Default is (1, 1), or lower-right corner.

size:Float

The "size" (scale) of the font.

smoothing:Bool

text:String

Text to display.

textColor:UInt

The color of the text in 0xAARRGGBB format.

read only textHeight:Float

Height of the text in this text field.

read only textWidth:Float

Width of the text in this text field.

updateImmediately:Bool

Whether graphics/bitmapdata of this text field should be updated immediately after each setter call. Default value is true which means that graphics will be updated/regenerated after each setter call, which could be CPU-heavy. So if you want to save some CPU resources then you could set updateImmediately to false, make all operations with this text field (change text color, size, border style, etc.). and then set updateImmediately back to true which will immediately update graphics of this text field.

useTextColor:Bool

Whether to use textColor while rendering or not.

wordWrap:Bool

A Boolean value that indicates whether the text field has word wrap.

wrapByWord:Bool

Whether word wrapping algorithm should wrap lines by words or by single character. Default value is true.
